== Description ==

Quick contact box is a nice widget that shows: 
* phone numbers 
* address
* email

in three different boxes.

Quick Contact Box is based on JavaScript / jQuery plugin that has a nice transition effect between a box and each other.

Quick Contact Box produces output valid xhtml and css
 
== Installation ==


1. Upload folder quick-contact-box to the `/wp-content/plugins/` directory
1. Activate the widget through the 'Plugins' menu in WordPress
1. Drag the widget in the sidebar and configure
